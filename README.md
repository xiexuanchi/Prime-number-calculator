# 质因数分解计算器

![界面截图](screenshot.png) <!-- 请手动添加程序截图 -->

基于PyQt6开发的图形化质因数分解工具，支持任意正整数分解计算。

## 功能特性
✅ 直观的GUI界面  
✅ 实时质因数分解计算  
✅ 输入有效性检测  
✅ 一键清空功能  
✅ 紫色主题配色方案  

## 使用说明
1. 在输入框输入大于1的正整数
2. 点击"分解"按钮获取质因数分解结果
3. 点击"清除"按钮重置所有内容

## 安装依赖
```bash
pip install PyQt6
```
